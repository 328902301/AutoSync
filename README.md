#### 自用配置：
* 自用配置：[Auto.conf](https://raw.githubusercontent.com/GodMoli/QuanX/main/File/Auto.conf) ❗❗
* 直接引用会覆盖原有配置，建议打开后直接复制相应字段使用，可自行DIY！
* 可借助fastgit为GitHub加速：```https://raw.githubusercontent.com/``` 替换为 ```https://raw.fastgit.org/```
