# 🍿️ DualSubs
  * Dual Subtitles for Streaming Media
  * 流媒体平台字幕增强及双语模块
  * **Powered by [Chavy](https://github.com/chavyleung)‘s [Env.js](https://github.com/chavyleung/scripts/blob/master/Env.js) and [BoxJs](https://chavyleung.gitbook.io/boxjs/)**
  * 需要启用`重写`、`脚本`、`MitM`功能
  * 有问题请至Issue页面反馈
