# 代理的导入
QR：

Q：什么是代理？

R：通俗来说就是你在机场购买的订阅链接

Q:订阅链接是什么？

R:就是一堆节点

Q:节点是什么？

R:一个节点代表一个代理服务器，代理服务器可以实现流量的中转。

PS.则订阅节点  是许多节点的集合，是一个配置在远端的节点集合，Loon通过url下载下来后解析成一个节点列表。

使用：打开软件于 [仪表]——[所有节点]——[右上角+]——[添加节点]
