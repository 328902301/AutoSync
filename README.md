# Clash_configuration

Clash For Windows的规则一直很麻烦，对于订阅的节点很多时候只需要它的节点信息就行，
mixin功能大概只能用来替换dns，虽然也可以替换规则。但是无法在UI显示，于是，使用parser是最好的选择

parser将将订阅结构完全统一化，订阅自带的规则组和规则完全丢弃而使用自己收集的
